# Branding ("Valkku") Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Vaihda sovelluksen nimi "Treeniapp"/"Treeni" → "Valkku" ja korvaa emoji-pohjainen PWA-ikoni itse piirretyllä SVG-ikonilla (abstrakti pulssiviiva), sekä lisää puuttuva favicon-linkki.

**Architecture:** Kaksi tiedostoa muuttuu: `index.html` (title, apple-mobile-web-app-title-meta, uusi favicon-link) ja `manifest.json` (name, short_name, icon-data-URI). Väriteema (musta+sininen) säilyy koskemattomana.

**Tech Stack:** Vanilla HTML/JSON, inline SVG data-URI, ei ulkoisia riippuvuuksia.

---

### Task 1: Nimenvaihto

**Files:**
- Modify: `index.html:10` (apple-mobile-web-app-title)
- Modify: `index.html:12` (title)
- Modify: `manifest.json:2-3` (name, short_name)

**Konteksti ennen muutosta** — `index.html` rivit 4-12:

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="theme-color" content="#000000">
<meta name="mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<meta name="apple-mobile-web-app-title" content="Treeniapp">
<link rel="manifest" href="manifest.json">
<title>Treeniapp</title>
```

- [ ] **Step 1: Päivitä `index.html`-nimet**

Korvaa yllä oleva lohko:

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="theme-color" content="#000000">
<meta name="mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<meta name="apple-mobile-web-app-title" content="Valkku">
<link rel="manifest" href="manifest.json">
<title>Valkku</title>
```

- [ ] **Step 2: Päivitä `manifest.json`-nimet**

Konteksti ennen muutosta:

```json
{
  "name": "Treeniapp",
  "short_name": "Treeni",
```

Korvaa:

```json
{
  "name": "Valkku",
  "short_name": "Valkku",
```

- [ ] **Step 3: Tarkista ettei muualla sovelluksessa esiinny nimeä**

```bash
grep -n "Treeniapp\|>Treeni<\|Treeni \|\"Treeni\"" index.html manifest.json
```

Odotettu tulos: ei osumia (jos osumia löytyy, ne pitää päivittää myös — mutta speksin mukaan sivupalkin versiochippi tai muu UI-teksti ei sisällä nimeä).

- [ ] **Step 4: Committaa**

```bash
git add index.html manifest.json
git commit -m "feat: nimenvaihto Treeniapp -> Valkku"
```

### Task 2: Uusi app-ikoni ja favicon

**Files:**
- Modify: `manifest.json:10-17` (icons-taulukko)
- Modify: `index.html` (uusi `<link rel="icon">` head-osioon)

**Konteksti ennen muutosta** — `manifest.json` rivit 10-17:

```json
  "icons": [
    {
      "src": "data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><rect width='100' height='100' fill='%230a84ff' rx='22'/><text y='68' x='50' text-anchor='middle' font-size='60'>🏋️</text></svg>",
      "sizes": "any",
      "type": "image/svg+xml",
      "purpose": "any maskable"
    }
  ]
```

- [ ] **Step 1: Päivitä `manifest.json`-ikoni**

Korvaa yllä oleva lohko:

```json
  "icons": [
    {
      "src": "data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><rect width='100' height='100' fill='%230a84ff' rx='22'/><path d='M10 54h16l8-28 12 56 12-40 8 12h24' fill='none' stroke='%23fff' stroke-width='9' stroke-linecap='round' stroke-linejoin='round'/></svg>",
      "sizes": "any",
      "type": "image/svg+xml",
      "purpose": "any maskable"
    }
  ]
```

- [ ] **Step 2: Lisää favicon `index.html`:iin**

Konteksti ennen muutosta (Task 1:n jälkeen):

```html
<link rel="manifest" href="manifest.json">
<title>Valkku</title>
```

Korvaa:

```html
<link rel="manifest" href="manifest.json">
<link rel="icon" type="image/svg+xml" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><rect width='100' height='100' fill='%230a84ff' rx='22'/><path d='M10 54h16l8-28 12 56 12-40 8 12h24' fill='none' stroke='%23fff' stroke-width='9' stroke-linecap='round' stroke-linejoin='round'/></svg>">
<title>Valkku</title>
```

- [ ] **Step 3: Validoi `manifest.json`**

```bash
python3 -m json.tool manifest.json
```

Odotettu tulos: tulostaa muotoillun JSON:in ilman virhettä (jos syntaksivirhe, komento heittää `json.decoder.JSONDecodeError`).

- [ ] **Step 4: Testaa manuaalisesti selaimessa**

Käynnistä paikallinen palvelin (`python3 -m http.server 8080`), avaa `http://localhost:8080/index.html`. Tarkista:
- Selaimen välilehden faviconissa näkyy sininen pyöristetty neliö valkoisella pulssiviivalla (ei tyhjä/oletusikoni).
- Välilehden otsikko on "Valkku".
- Konsolissa ei virheitä (`read_console_messages`, pattern `error|Error`).

- [ ] **Step 5: Committaa**

```bash
git add index.html manifest.json
git commit -m "feat: uusi app-ikoni ja favicon (pulssiviiva)"
```
